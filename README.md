# AI-Based-Career-Guidance-Chatbot
AI-Based Smart Career Guidance Chatbot developed using Python, Streamlit, and SQLite. The system provides personalized career recommendations, resume analysis, personality assessment, skill-gap detection, internship suggestions, and a structured career roadmap using a rule-based expert system and NLP concepts.
# AI-Based Smart Career Guidance Chatbot

## Overview
An AI-powered career guidance and decision support system that provides personalized career recommendations, resume analysis, personality assessment, skill-gap detection, internship suggestions, and career roadmap generation.

## Technologies Used
- Python
- Streamlit
- SQLite
- NLP
- Expert System

## Features
- Career Recommendation Engine
- Resume Analysis
- Personality Assessment
- Skill Gap Detection
- Career Roadmap Generator
- Admin Dashboard
